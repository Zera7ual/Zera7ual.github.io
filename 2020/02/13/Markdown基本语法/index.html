Markdown基本语法

标题目录

分级标题

# 表示一级标题，## 表示二级标题，以此类推

内容目录

使用 [TOC] 显示目录结构

文字

斜体和粗体

使用 * 和 ** 表示斜体和粗体

示例：

这是 斜体，这是 粗体。

删除线

使用 ~~ 表示删除线。

这是一段错误的文本。

列表

无序列表

使用 *，+，- 表示无序列表。

示例：

  • 无序列表项 一
  • 无序列表项 二
  • 无序列表项 三

有序列表

使用数字和点表示有序列表。

示例：

  1. 有序列表项 一
  2. 有序列表项 二
  3. 有序列表项 三

待办事宜

使用带有 [ ] 或 [x] （未完成或已完成）项的列表语法撰写一个待办事宜列表，并且支持子列表嵌套以及混用Markdown语法，例如：

对应显示如下待办事宜 Todo 列表：

  • 笔记
    • 编写
      • 支持 MD
      • 支持代码高亮
    • 存放和阅读
      • 支持手机端
    • 整理
      • 一本书完成后，整理至GB

引用链接

文字引用

使用 > 表示文字引用。

示例：

别放弃你的梦，继续睡吧。

外链接

使用 [描述](链接地址) 为文字增加外链接。

示例：Linux命令大全

代码

行内代码块

使用 `代码` 表示行内代码块。

示例：

让我们聊聊 Linux。

代码块

使用 四个缩进空格 表示代码块。

示例：

这是一个代码块，此行左侧有四个不可见的空格。

Html 标签

可以在 Markdown 语法中嵌套 Html 标签，你可以用 Html 写一个纵跨两行的表格：

<table>
    <tr>
        <th rowspan="2">值班人员</th>
        <th>星期一</th>
        <th>星期二</th>
        <th>星期三</th>
    </tr>
    <tr>
        <td>李强</td>
        <td>张明</td>
        <td>王平</td>
    </tr>
</table>
值班人员 星期一 星期二 星期三
李强 张明 王平
